Each themed primitive is one inline-positioned SVG dropped into the scene via PLACE { element: <svg-id>, position: ... }. No new gesture, no new stage, no schema change — the existing PLACE machinery accepts arbitrary element ids whose markup is supplied via an asset: map at the top of the YAML (or per-scene). Promote to first-class stage only when authoring repeatedly needs the same primitive.
